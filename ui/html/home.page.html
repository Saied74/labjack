Application Summary

This summary is not a substitute for reading the U3 documentaton. It is only a summary of what is implemented in this program which is a limited set of LabJack U3 functions. LabJack U3 is a delicate instrument and a number of percautions must be excercised while using it.

This project is implemented in two phases

  • Phase 1: Analog and digital inputs and outputs
  • Phase 2: Timers and counters
    •

    Timers and counters implement many simple and complex functions and after a review of the needs, specific functions will be built

    High Voltage Analog Inputs

    Pins labled AIN0 through AIN3 are permanently set to high voltage Analog inputs (0-10 volts) and are not programmable.

    Flexible I/O Lines

    Lines FIO4 through FIO7 (on the screw terminals) and lines EIO0 through EIO7 (on DB15 connector) are flexible IO pins and can be programmed as digital input, digital output, or analog input channels. See the link "Configure U3" on the navigation bar on top of this page.

    Dedicated Digital IO Lines

    Lines CIO0 through CIO3 (on the DB25 connector) are digital only pins. They can be prgrammed as digital input or output. See the link "Configure U3" on the navigation bar on top of this page).
